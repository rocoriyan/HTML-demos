Change image on hover

This is can be done fairly easily by manipulating the DOM with JavaScript!

No hover

The code for this looks like this:

<!DOCTYPE html>
<html lang="en">
    <body>
    <div id="image-container">
        <img src="media/Glenn-1.png"/>
        <p>No hover</p>
    </div>
    <script>
        const imageToChange = document.querySelector("#image-container img");

        imageToChange.addEventListener('mouseover', () => {
            imageToChange.src = "media/Glenn-2.png";
            document.querySelector("#image-container p").innerText = "Hover";
        });

        imageToChange.addEventListener('mouseout', () => {
            imageToChange.src = "media/Glenn-1.png";
            document.querySelector("#image-container p").innerText = "No hover";
        })
    </script>
    </body>
</html>

mouseover is used to change the image when a mouse is hovering over the image, while mouseout is used to change the image back when the mouse stops hovering.

querySelector() and getElementById() are very similar methods. I've used document.querySelector here as it lets you to select elements using CSS selectors, allowing me to select the image and paragraph tag inside the image-container div, despite them not having their own IDs. getElementById is generally preferred, though, as it's faster. You can also only select one element, anyway, so selecting elements like this can be messy. If multiple elements match the CSS query used for document.querySelector, only the first one will be changed.

You might want to do more reading into DOM manipulation to see what works best for you. If you want to see what else can be done with DOM manipulation, I have a different website showing off the some examples.
