Change image on hover

This is can be done fairly easily by manipulating the DOM with JavaScript!

No hover

The code for this looks like this:

<!DOCTYPE html>
<html lang="en">
    <body>
    <div class="domedit image-container">
        <img src="media/Glenn-1.png"/>
        <p>No hover</p>
    </div>
    <script>
        const domDiv = document.querySelector(".domedit");

        domDiv.addEventListener('mouseover', () => {
            document.querySelector(".domedit img").src = "media/Glenn-2.png";
            document.querySelector(".domedit p").innerText = "Hover";
        });

        domDiv.addEventListener('mouseout', () => {
            document.querySelector(".domedit img").src = "media/Glenn-1.png";
            document.querySelector(".domedit p").innerText = "No hover";
        })
    </script>
    </body>
</html>

I've used document.querySelector here as I wanted to manipulate both the contents of the paragraph and the image's source. document.getElementsByTagName is much faster, though, it doesn't let you select elements by class.

You might want to do more reading into DOM manipulation to see what works best for you. If you want to see what else can be done with DOM manipulation, I have a different website showing off the some examples.
